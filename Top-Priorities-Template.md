# Top Priorities Template

Copy this into your Google Doc: **My Drive > Granola > Top Priorities**

Replace the examples with your actual priorities. The agent reads this every morning to frame your daily brief -- what's listed here determines what gets flagged as "Top Priority" vs. "Important" in your Slack DM.

---

## How to use this doc

- Keep 5-7 items max. If everything is a priority, nothing is.
- The agent will update this doc if a meeting reveals a priority shift (new deal heating up, deal going cold, leadership directive). It will always add a dated note explaining the change.
- You can also edit this doc yourself anytime. The agent reads whatever is current.
- Items at the top of the list are higher priority than items at the bottom.
