Workstream 1

GTM Discovery & Process Documentation

  • Document current vs. target GTM motion (AE-led, IPM-driven, SPICED-aligned)
  • Define sales stages gated by SPICED criteria (Stage 3 = Impact + Critical Event + Decision)
  • Build discovery playbooks tied to each SPICED element
  • Set activity expectations: 8+ client meetings/week, outbound minimums, in-meeting cadence
  • Articulate ICP and buyer personas from 18-month win data
  • Map territory methodology (80% geo, 20% vertical SMEs)
Workstream 2

CRM Implementation & Reporting

  • Configure SPICED-aligned deal stages with gating criteria in HubSpot
  • Implement buying committee mapping (EB, Champion, Tech Validator) on deals
  • Build ICP-score fields benchmarked against 18-month win-rate data
  • Set up activity tracking: emails, calls, meetings, notes
  • Create management dashboards: IPMs/AE, IPM-to-Pipeline, win rate by ICP
  • Prepare Alysio data-model handoff for predictive forecasting
Workstream 3

Data Enrichment & Marketing Alignment

  • Configure company enrichment (size, tech-stack, traffic signals) via BuiltWith, Clay, Similarweb
  • Define lead scoring logic: ICP fit + activity history + engagement signals
  • Clarify MQL → AE-owned handoff rules surfaced in reporting
  • Align marketing and sales on lead lifecycle definitions
Workstream 4

RevOps Tech Stack Review

  • Inventory GTM stack: HubSpot, Alysio, NotebookLM/Gemini, automation tools
  • Assess data quality, field consistency, and automation gaps
  • Identify opportunities to reduce manual data entry and improve seller hygiene
  • Deliver RevOps roadmap: AI readiness, roles/resourcing, 30-day implementation plan
